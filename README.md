<h2 id="日本語">日本語</h2>

### DeskGG？

デスクトップがアイコンでいっぱいになってしまったことはありませんか？<br>
デスクトップを綺麗にしたい、ジャンルを分けてショートカットを管理したい人にDeskGGはまさに適しています！<br>
壁紙を見せたい人からミニマリストまで、DeskGGはきっと気にいるでしょう.....？

### 機能

- 最大60アプリをフォルダーに収納できます
- アプリフォルダーの色、名前を変更できます
- アプリフォルダー内のアイコンをドラッグして順番を変更できます
- アプリフォルダー内にあるショートカットは右クリックメニューに対応しています
- アプリの利用に料金は必要ありません！完全無料でご利用いただけます。もしよければ[Ko-Fi](https://ko-fi.com/rec877dev)で寄付をお願いします.....

### インストール

- [Release](https://github.com/Banana-Niku87729/DeskGG/releases/)からインストーラーを入手し、実行します。<br>
- [Release](https://github.com/Banana-Niku87729/DeskGG/releases/)からポータブルを入手し、zipを解凍したあとstartフォルダにあるlauncher.exeを起動します。
- あるいは.....自分でビルドします。

### ビルド

[ここ](https://download-directory.github.io/?url=https://github.com/Banana-Niku87729/DeskGG/tree/main/app/main/1.0.0)から、ソースコードをダウンロードして、散らかっても問題ないフォルダに解凍します。<br>
そのディレクトリのコマンドプロンプトを開き、`dotnet build -c Release`を実行します。<br>
launcher、updater、Settingフォルダすべてに同じようにビルドします。
ルートディレクトリのbinフォルダに出力されたexeファイルと、依存ファイルをすべて移動します。<br>
あるいは、好きなフォルダに移動します.....

<h2 id="english">English</h2>

### What is DeskGG?

Have you ever found your desktop cluttered with endless icons?<br>
DeskGG is perfect for anyone who wants to clean up their desktop or organize shortcuts by category!<br>
From wallpaper showcase enthusiasts to minimalists, DeskGG is sure to fit your style... right?

### Features

- Store up to 60 apps in a single folder
- Customize folder colors and names
- Rearrange icons inside folders by dragging and dropping
- Shortcuts inside folders support context (right-click) menus
- 100% free with no costs involved! If you like it, consider supporting via [Ko-Fi](https://ko-fi.com/rec877dev)...

### Installation

- Download the installer from [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/) and run it.<br>
- Download the portable zip from [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/), extract it, and run `launcher.exe` inside the `start` folder.
- Or... build it yourself!

### Building

Download the source code from [here](https://download-directory.github.io/?url=https://github.com/Banana-Niku87729/DeskGG/tree/main/app/main/1.0.0) and extract it to a workspace folder.<br>
Open Command Prompt in that directory and run `dotnet build -c Release`.<br>
Repeat this build process for the `launcher`, `updater`, and `Setting` folders.
Move all compiled exe files and dependency files from the root `bin` folder to your preferred directory.

### Usage

1. Launch `launcher.exe`<br>
2. Right-click the DeskGG icon in the taskbar / system tray<br>
3. Create a new folder<br>
4. A new app folder will appear on your screen<br>
5. Drag and drop shortcuts into the newly created folder to organize them!

<h2 id="한국어">한국어</h2>

### DeskGG란?

바탕화면이 아이콘으로 가득 차서 지저분했던 적이 있으신가요?<br>
DeskGG는 바탕화면을 깔끔하게 정리하고, 카테고리별로 바로가기를 관리하고 싶은 분들에게 최적의 도구입니다!<br>
배경화면을 시원하게 보고 싶은 분부터 미니멀리스트까지, DeskGG가 맘에 드실 거예요... 아마도?

### 주요 기능

- 하나의 폴더에 최대 60개의 앱 보관 가능
- 앱 폴더의 색상 및 이름 변경 기능
- 드래그 앤 드롭으로 폴더 내 아이콘 순서 변경 가능
- 폴더 안의 바로가기 아이콘 우클릭 메뉴(컨텍스트 메뉴) 지원
- 완전히 무료로 이용 가능합니다! 마음에 드신다면 [Ko-Fi](https://ko-fi.com/rec877dev)를 통해 후원해 주시면 감사하겠습니다...

### 설치 방법

- [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/)에서 설치 파일(Installer)을 다운로드하여 실행합니다.<br>
- [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/)에서 포터블(Portable) zip 파일을 받아 압축을 푼 뒤, `start` 폴더 안의 `launcher.exe`를 실행합니다.
- 아니면... 직접 빌드합니다!

### 빌드 방법

[여기](https://download-directory.github.io/?url=https://github.com/Banana-Niku87729/DeskGG/tree/main/app/main/1.0.0)에서 소스 코드를 다운로드하고 임의의 폴더에 압축을 풉니다.<br>
해당 디렉토리에서 명령 프롬프트를 열고 `dotnet build -c Release`를 실행합니다.<br>
`launcher`, `updater`, `Setting` 폴더에 대해 동일하게 빌드를 진행합니다.
루트 디렉토리의 `bin` 폴더에 출력된 exe 파일과 의존성 파일들을 필요한 위치로 이동시킵니다.

### 사용 방법

1. `launcher.exe`를 실행합니다.<br>
2. 작업 표시줄(트레이)에 추가된 아이콘을 우클릭합니다.<br>
3. 새 폴더를 생성합니다.<br>
4. 새로운 앱 폴더가 생성됩니다.<br>
5. 생성된 폴더로 추가하고 싶은 바로가기 아이콘을 드래그 앤 드롭합니다.

<h2 id="繁體中文">繁體中文</h2>

### 關於 DeskGG

您的桌面是否曾經塞滿了各式各樣的圖示？<br>
對於想要整理桌面、將捷徑按類別分類管理的人來說，DeskGG 是再適合不過的工具！<br>
無論是想展現精美桌布的人，還是極簡主義者，相信都會喜歡 DeskGG.....吧？

### 主要功能

- 單一資料夾最多可收納 60 個應用程式
- 可自訂應用程式資料夾的顏色與名稱
- 支援拖曳資料夾內的圖示以調整排序
- 資料夾內的捷徑支援右鍵選單功能
- 完全免費使用！如果您喜歡這個專案，歡迎透過 [Ko-Fi](https://ko-fi.com/rec877dev) 給予贊助支持.....

### 安裝方式

- 從 [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/) 下載安裝檔並執行。<br>
- 從 [Releases](https://github.com/Banana-Niku87729/DeskGG/releases/) 下載可攜版 (Portable) zip 檔，解壓縮後執行 `start` 資料夾裡面的 `launcher.exe`。
- 或者.....選擇自行編譯。

### 編譯方法

點擊 [此處](https://download-directory.github.io/?url=https://github.com/Banana-Niku87729/DeskGG/tree/main/app/main/1.0.0) 下載原始碼，並解鎖至任意工作資料夾。<br>
在該目錄下開啟命令提示字元 (CMD)，執行 `dotnet build -c Release`。<br>
請針對 `launcher`、`updater` 及 `Setting` 資料夾進行相同的編譯動作。
最後將根目錄 `bin` 資料夾中輸出的 exe 檔案與相關相依檔案移動至您指定的資料夾即可。

### 使用方法

1. 啟動 `launcher.exe`<br>
2. 右鍵點擊系統工作列（Tasktray）上的 DeskGG 圖示<br>
3. 建立新資料夾<br>
4. 畫面上將會出現新的應用程式資料夾<br>
5. 將想要整理的捷徑直接拖曳至該資料夾中即可
