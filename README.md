# DeskGG

Clean up your windows desktop.

</div>

## DeskGG？

デスクトップがアイコンでいっぱいになってしまったことはありませんか？<br>
デスクトップを綺麗にしたい、ジャンルを分けてショートカットを管理したい人にDeskGGはまさに適しています！<br>
壁紙を見せたい人からミニマリストまで、DeskGGはきっと気にいるでしょう.....？

## 機能

- 最大60アプリをフォルダーに収納できます
- アプリフォルダーの色、名前を変更できます
- アプリフォルダー内のアイコンをドラッグして順番を変更できます
- アプリフォルダー内にあるショートカットは右クリックメニューに対応しています
- アプリの利用に料金は必要ありません！完全無料でご利用いただけます。もしよければ[Ko-Fi](https://ko-fi.com/rec877dev)で寄付をお願いします.....

## インストール

- [Release](https://github.com/Banana-Niku87729/DeskGG/releases/)からインストーラーを入手し、実行します。<br>
- [Release](https://github.com/Banana-Niku87729/DeskGG/releases/)からポータブルを入手し、zipを回答したあとstartフォルダにあるlauncher.exeを起動します。
- あるいは.....自分でビルドします。

## ビルド

[ここ](https://download-directory.github.io/?url=https://github.com/Banana-Niku87729/DeskGG/tree/main/app/main/1.0.0)から、ソースコードをダウンロードして、散らかっても問題ないフォルダに解凍します。<br>
そのディレクトリのコマンドプロンプトを開き、```dotnet build -c Release```を実行します。<br>
launcher、updater、Settingフォルダすべてに同じようにビルドします。
ルートディレクトリのbinフォルダに出力されたexeファイルと、依存ファイルをすべて移動します。<br>
あるいは、好きなフォルダに移動します.....

## 使い方

![launcher.exe起動](https://github.com/Banana-Niku87729/DeskGG/blob/main/assets/lunch_deskgg_launcher.png?raw=true)<br>

1. launcher.exeを起動します<br>

![タスクトレイ](https://github.com/Banana-Niku87729/DeskGG/blob/main/assets/tasktray_right_click.png?raw=true)<br>

2. タスクトレイに追加されたアイコンを右クリックします<br>

![フォルダー作成](https://github.com/Banana-Niku87729/DeskGG/blob/main/assets/tasktray_right_click_create_folder.png?raw=true)<br>

3. 新規フォルダーを作成します<br>

![新しいフォルダー](https://github.com/Banana-Niku87729/DeskGG/blob/main/assets/new_folder_deskgg.png?raw=true)<br>

4. 新しいフォルダーが作成されます<br>

![フォルダーにアプリを追加](https://github.com/Banana-Niku87729/DeskGG/blob/main/assets/deskgg_appfolder_addshortcut_preview.gif?raw=true)<br>

5. 作成されたアプリフォルダに追加したいショートカットをドラッグアンドドロップします
